# Contact Management System

A simple command-line application built with Python to manage a list of contacts. This project allows users to add, view, search, and delete contacts with error handling for invalid inputs.

## Features
- **Add Contact**: Add a new contact with a name and phone number, with duplicate name checking.
- **View Contacts**: Display all contacts in a numbered list with total count.
- **Search Contact**: Search for a contact by name (case-insensitive).
- **Delete Contact**: Remove a contact by name.
- **Error Handling**: Validates phone numbers (must be digits) and menu choices (1-5).
- **User-Friendly Interface**: Simple menu-driven CLI for easy interaction.
